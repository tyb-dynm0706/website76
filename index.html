GitHub Pagesメモ

新規websiteの作り方

設定編

  1. Home ＞ 「NEW」 をクリック
  2. 1.General ＞ Repository nameにフォルダ名となるものを入力（ここではwebsite76と入力した）
  3. 2.Configuration ＞ 「Choose visibility」 ＞ 「Public（公開）」にチェック。
    ＊privateでも後ほど変更できる。
  4. 「Create repository」ボタンを押す

HTMLファイルを作る編

  1. 1.青枠の中にある「creating a new file」というテキストをクリック
  2. 2.上部のファイル名に「index.html」を入力
    例　website76/「index.html」in main
  3. htmlを書く
  4. 3.右上にある緑色の「Commit changes...」ボタンを押す
  5. 「Commit changes」ボタンを押す

公開編設定編

  1. 1.「Settings」タブを開く
  2. 2.左メニューにある「Pages」を選択
  3. 3.「Build and deployment」 ＞ 「Branch」 ＞ 「None」→「main」変え「Save」ボタンを押す
  4. 3.右上にある緑色の「Commit changes...」ボタンを押す
